Стоимость проекта определяется совокупностью стоимостей ресурсов, стоимостью и временем выполнения работ проекта [cite: mazur, стр. 590]. Бюджет -- это реестр планируемых расходов и доходов с распределением по статьям на соответствующий период времени, определяющий ресурсные ограничения проекта [cite: mazur, стр. 592]. Смета проекта -- документ, содержащий обоснование и расчёт стоимости проекта (контракта), как правило, на основе объёмов работ проекта, требуемых ресурсов и цен [cite: mazur, стр. 592]. После согласования на основе сметы формируется бюджет.

Заказчик может финансировать и контролировать выполнение проекта на основе сметы. Смета содержит перечень работ, сгруппированных по области, виду деятельности или другими способами, со сроками и стоимостью работ, а также общий срок и общую стоимость проекта.

Все затраты по проекту можно классифицировать:

- как прямые и накладные расходы;
- повторяющиеся и единовременные;
- постоянные и переменные по признаку зависимости от объёма работ;
- плату за сверхурочное и нормальное рабочее время.

Бюджетирование проекта -- определение стоимостных значений выполняемых в рамках проекта работ и проекта в целом; процесс формирования бюджета, описывающего распределение затрат по статьям, видам работ, по времени их выполнения или по иной структуре [cite: mazur, стр. 601].

При контроле стоимости проекта учитываются следующие показатели [cite: mazur, стр. 606-607]:

- затраты, необходимые для завершения (НДЗ) -- сколько средств ещё нужно вложить, чтобы завершить работу;
- расчётная стоимость -- общая сумма затрат при завершении работы, равна сумме фактических и НДЗ затрат.

Расхождение по затратам -- разница между фактическими и плановыми затратами [cite: mazur, стр. 607].

Метод освоенного объёма основан на определении отношения фактических затрат к объёму работ, которые должны быть выполнены к определённой дате. Выявленные тенденции используются для прогноза будущей стоимости работ при завершении проекта и для определения факторов, оказывающих влияние на график выполнения работ. [cite: mazur, стр. 607-609]

Освоенный объём -- плановая стоимость фактически выполненных работ.

По затратам [cite: mazur, стр. 613]:  
CV (Cost variance) - отклонение в денежных единицах, равно разнице между освоенным объёмом и фактическими затратами.  
CVP - отклонение в процентах, равно 100% * CV / освоенный_объём.

По расписанию [cite: mazur, стр. 613]:  
SV (Schedule variance) - отклонение в денежных единицах, равно разнице между освоенным объёмом и плановыми затратами.  
SVP - отклонение в процентах, равно 100% * SV / освоенный_объём.

Индекс освоения затрат [cite: mazur, стр. 611]: освоенный_объём / фактические_затраты.  
Индекс выполнения плана [cite: mazur, стр. 611]: освоенный_объём / плановые_затраты.

Оценка конечной стоимости проекта [cite: mazur, стр. 613]: плановые_затраты * индекс_освоения_затрат.
